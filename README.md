## 🧬 核心：七维分析框架

经过**4轮强化学习训练**，Agent已形成稳定的分析模板：

| 维度 | 内容 | 关键方法 |
|------|------|----------|
| 🧬 **生意本质** | 卖的到底是什么？不是产品，是本质 | 跨学科视角（生物学/心理学/物理学） |
| 🏰 **护城河** | 五种类型判断 + 竞品对比 | "100亿美元+优秀CEO能否复制？" |
| 👨‍💼 **管理层审计** | 10年言行一致性 | 红线——有问题多便宜都不投 |
| 💰 **股东回报** | 分红/回购/留存利润效率 | "一元前提"检验 |
| 🔄 **逆向思考** | 3-5条真实失败路径 | "告诉我我会死在哪里" |
| 📈 **盈亏同源** | 赚和亏来自同一个东西 | 这个"源"是什么？ |
| 🎯 **估值击球点** | 精确数字 | 买入价 ≥ 30%安全边际 + 卖出条件 |

### 🟢🟡🔴 输出示例

```
| 维度     | 评级 | 一句话                    |
|----------|:----:|--------------------------|
| 能力圈   |  🟢  | 简单可理解的消费品生意      |
| 护城河   |  🟢  | 品牌+分销网络不可复制       |
| 管理层   |  🟡  | 能力尚可，但过度扩张隐忧     |
| 股东回报 |  🟢  | 连续10年分红增长            |
| 估值     |  🟡  | 合理不便宜，等待击球点       |
| 最终判断 |  🟡  | 观察——等安全边际足够再挥棒   |
```
