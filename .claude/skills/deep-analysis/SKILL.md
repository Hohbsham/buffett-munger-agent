# 深度公司分析

使用经过4轮强化学习训练的完整七维分析框架。

## 强制输出模板

### 一、生意的底层逻辑
- 这家公司卖的到底是什么？不是产品，是**本质**。
- 芒格式追问：如果只能用一个词描述这门生意的核心，是什么？
- 必须包含至少一个跨学科视角（生物学/心理学/物理学/历史学）。

### 二、护城河分析
- 五种护城河类型判断 + 竞品对比
- 护城河深度测试：100亿美元+优秀CEO能否复制？
- 未来10年最可能的侵蚀因素

### 三、管理层人品审计
- 10年言行一致性检查
- 年报坦诚度
- 资本配置的理性程度
- 薪酬与持股结构
- 红牌信号检查

### 四、股东回报
- 分红率、回购记录、留存利润使用效率
- "一元前提"检验

### 五、逆向思考——怎么死？
- 3-5条失败路径，深入分析
- 最可能的死法是什么？

### 六、当前市场环境 + 盈亏同源
- 结合宏观/行业现状
- 盈亏同源：赚的钱和亏的钱来自同一个东西——这个"源"是什么？

### 七、估值与击球点
- 🎯 击球点（买入价）：安全边际 ≥ 30%
- 🚨 卖出点：什么价格/条件必须卖？
- 给出具体数字

## 分析标准
- 至少引用一句巴菲特股东信或芒格演讲原话
- 必须包含反面论证（逆向思考）
- 不假装精确，但给出具体数字
- 如果公司在他的能力圈之外，直接说"太难了"并解释原因
